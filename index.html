Company Directory

The Company Website demonstrates CRUD functionality which will allow for the creation, deletion and the updating of sections within a fictional Company Website from a MySQL Database.

In the top left corner of the page there are three buttons, the first button is the Employee Button, then the Department Button which is followed by the Location Button. Depressing each button will display the corresponding screen which will show the number for each section in card form.

The Add Button at the top of the screen allows for the Addition of the Employees, Departments and Locations respectively. While the Delete and Edit Buttons on the cards themselves will delete or edit that particular entry.

The depressing of the Add and Edit Buttons will show a Modal that will show the user the data that can be changed. The Add Modal requires all fields to be filled except the Job Title field as it is empty for all Employees in the Database.

Department and Location Entries can not be deleted unless the dependencies they are associated with are deleted first, if the delete button is depressed a message stating either to confirm the deletion of a particular entry or showing the number of dependents associated with that particular entry.
